⚙️ Setup Guide

        
Real ML detection ke liye Plant.id API setup karein

      


      

        
🔑 Free API Key Kaise Milegi

        
Web.plant.id par jayen: web.plant.id

        
Free account banayein (email se)

        
Dashboard → API Keys → Create New Key

        
Key copy karein aur Scan tab mein paste karein

        
Free plan: 100 identifications/month

      


      

        
1

        

          
Tasveer Upload

          
Camera ya gallery se flower/plant ki tasveer lein. App seedha base64 mein convert karta hai.

        

      


      

        
2

        

          
Plant.id ML API

          
Tasveer Plant.id ke AI server par jaati hai jo 20,000+ plant species ko pehchanta hai.

          POST api.plant.id/v2/identify
        

      


      

        
3

        

          
Wikipedia Data

          
Plant identify hone ke baad Wikipedia API se detailed information fetch hoti hai automatically.

          GET en.wikipedia.org/api/rest_v1/
        

      


      

        
4

        

          
Results Display

          
Confidence score, scientific name, common names, aur care tips — sab real data se.
